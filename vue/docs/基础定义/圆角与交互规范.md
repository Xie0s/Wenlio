# 圆角与交互规范

## 背景与目标

为统一 `src/components/ui` 目录下组件的视觉语言，本规范定义了圆角分层和关键交互组件的按钮约束。

## 规范项

### 1) 容器级圆角（`rounded-2xl`）

所有顶层容器、卡片等统一使用 `rounded-2xl`（1rem）。

- 代表组件：`Card`、`DropdownMenuContent`、`SelectContent`、`ContextMenuContent`、`DrawerContent`、`HoverCardContent`、`Input`、`Textarea` 等。

### 2) 超大圆角（`rounded-3xl`）

特殊浮层、模态框使用 `rounded-3xl`（1.5rem）。

- 代表组件：`Dialog`、`AlertDialog`、`PopoverContent` 等。

### 3) 交互元素圆角（`rounded-xl`）

容器内部交互项或辅助装饰元素统一使用 `rounded-xl`（0.75rem）。

- 代表组件：`DropdownMenuItem`、`SelectContent` 内部项、`CommandItem`、`MenubarItem`、`Dialog` 关闭按钮等。
- 特殊说明：`TooltipContent` 保持 `rounded-md`。

### 4) 交互进阶方案

#### Dialog / DialogScrollContent

- 右上角按钮默认使用 `rounded-full` 圆形轮廓样式。
- 支持 `:show-close-button="false"` 隐藏关闭按钮。
- 内置操作按钮支持 Tooltip 文字提示。

#### AlertDialog（Action / Cancel）

- 取消操作固定为 `X` 图标。
- 确认操作固定为 `Check` 图标。
- 确认按钮样式：`primary + rounded-full`。
- 取消按钮样式：`outline + rounded-full`。

## 示例（类名分层示意）

- 容器层：`rounded-2xl`
- 模态层：`rounded-3xl`
- 交互层：`rounded-xl`
- 例外项：`TooltipContent` 使用 `rounded-md`

## 维护建议

- 新增组件时先判断所属层级，再选择对应圆角，避免同层级组件风格漂移。
- 若出现视觉冲突，优先在组件规范中登记例外项，不在页面层临时覆写。
