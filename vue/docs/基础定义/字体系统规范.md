# 字体系统规范

## 背景与目标

项目统一使用 `AlibabaPuHuiTi` 作为全局默认字体，目标是在不同页面和组件中保持一致的字体表现，避免局部字体回退不一致。

## 规范项

### 1) 字体资源规范

- 字体文件目录：`/public/fonts/`
- 当前字重覆盖：
  - Light（300）
  - Regular（400）
  - Medium（500）

### 2) 接入实现规范

- 在 `src/style.css` 中通过 `@font-face` 定义字体家族。
- 在 `:root` 中定义变量：`--font-sans: "AlibabaPuHuiTi"`。
- 在 `@theme` 中通过 `--font-sans` 注入 Tailwind 字体系统。
- 在 `base` 层统一应用字体家族，确保全站一致。

## 示例（结构示意）

```css
/* src/style.css */
@font-face { /* 省略具体 src 与字重 */ }

:root {
  --font-sans: "AlibabaPuHuiTi";
}

@theme {
  --font-sans: var(--font-sans);
}

@layer base {
  body {
    font-family: var(--font-sans);
  }
}
```

## 维护建议

- 新增字重时，按现有目录规范补充到 `/public/fonts/`，并同步完善 `@font-face`。
- 如出现局部字体异常，优先排查是否绕过了 `--font-sans` 或覆盖了 `base` 层样式。
